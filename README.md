##  Overview

Simple E-Commerce Store is a basic web application where users can view products and add items to cart.  
This project is created using HTML, CSS, and JavaScript.

##  Features

-  Product display
-  Add products to cart
-  Total price calculation
-  Simple user interface

## Technologies Used

- HTML5
- CSS3
- Javascript 

##  How to Run

1. Download the project.
2. Open `index.html` in a browser.
3. Use the shopping features.

##  Objective

To build a simple e-commerce interface and understand frontend development concepts.
